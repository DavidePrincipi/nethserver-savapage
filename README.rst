========
SavaPage
========

Install and configure SavaPage: https://www.savapage.org/

The package:

- install SavaPage installer as /usr/local/bin/savapage-setup-linux-x64.bin


Database: ::

 savapage=service
    PaperSize=a4
    TCPPorts=8631,8632,8639,5353,5222
    UDPPorts=8631,8632,8639,5353,5222
    access=green
    status=enabled
